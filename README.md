## 品牌简介

**寻心理 · InnerSeed** 是一个由心理学硕博团队创立的品牌，致力于把心理学从课本和咨询室里带出来，让它走进日常生活。

- **品牌标语**：向内寻，向外生
- **核心理念**：每个人心里都有一颗种子，向内寻是找到它，向外生是让它发芽
- **业务范围**：心理团辅课程、心理学文创产品、课程培训
